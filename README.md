## 📃 개요
### 소개
- 쇼핑몰 '뮤직포스' 사이트를 클론한 개인 프로젝트 <br>
- 원본 쇼핑몰의 상품 정보는 데이터를 크롤링하여 사용

### 프로젝트 목적
- 웹 전반에 대한 학습과 이해도를 높이기 위해 풀스택으로 온라인 쇼핑몰을 클론하는 프로젝트를 진행 <br>

### 개발기간
- 25년 2월 ~ 진행중 <br>

### 도메인
- [hyun-clone.shop](http://hyun-clone.shop)

### 원본 사이트
- [뮤직포스 홈페이지](https://musicforce.co.kr/index.html)
